# 复合数据类型

## 元组 tup

多个不同数据类型的合集，可以放置不同的数据类型，定义之后不可修改。

使用 .0，.1 来访问第一个，第二个元素

## 数组 Array

存储相同数据类型的元素，也不能修改，还是使用最常用的 [0]，[1] 来访问元素，因为其不可修改的因素，使用应该不会太高。
